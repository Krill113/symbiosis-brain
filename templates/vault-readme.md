# Symbiosis Brain — Vault

Это твоя общая память с Claude. Markdown-файлы, которые Claude может читать,
писать и связывать через `[[wiki-links]]`.

## Структура

| Папка | Что хранится |
|---|---|
| `projects/` | Карточки конкретных проектов |
| `wiki/` | Долгоживущая справочная информация |
| `decisions/` | Зафиксированные решения с обоснованием |
| `patterns/` | Повторяющиеся приёмы и шаблоны |
| `mistakes/` | Ошибки прошлого с уроками |
| `feedback/` | Правила работы (от тебя — для Claude) |
| `research/` | Исследования и обзоры |
| `reference/` | Канонические таблицы (например, `scope-taxonomy.md`) |

## Scope (область видимости)

Каждая заметка имеет `scope:` во frontmatter — `global` или конкретный проект.
Сессии видят свой scope + `global`. Список валидных scope-ов:
`reference/scope-taxonomy.md`. Новый scope регистрирует skill `brain-project-init`.

## Wiki-ссылки

`[[projects/foo]]` или `[[user/some-rule]]` создают граф связей. Откроется
красиво в Obsidian, работает и без него.

## Команды

- `/brain-recall` — позвать память на конкретную задачу
- `/brain-save` — сохранить вывод/решение/урок
- `/brain-help` — подробности

## Git-sync

Обычная папка с markdown — можешь `git init`, пушить на GitHub,
синхронизировать между машинами.
